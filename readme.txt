для работы необходима библиотека pygame

Путь к запуску:
	n!/RSP.py
В коде указаны характеристики которые можно менять:

- #colors:
	изменить цвет бэкграунда
- #screen & image settings
	изменить размер окна
	изменить размер объектов(камень, ножницы, бумага)
- #object settings
	изменить изначальную скорость объектов по осям X и Y
	изменить промежуток цифр от куда будет браться рандомное количество объектов
	по желанию можно поставить статичное количество объектов

правильные расположения файлов:
    n!/images/paper.png
    n!/images/rock.png
    n!/images/scissors.png
    n!/readme.txt
    n!/RSP.py
    n!/sounds/paper.mp3
    n!/sounds/rock.mp3
    n!/sounds/scissors.mp3
